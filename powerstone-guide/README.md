# Power Stone Guide - パワーストーンガイドブック

パワーストーンの説明書として作成された、ラグジュアリーなウェブサイトです。

## デザイン

**Art Deco Revival（アールデコ・リバイバル）**
- カラー: リッチブラック × アンティークゴールド × ディープネイビー
- 幾何学的なパターンと放射状のサンバーストモチーフ
- スムーズなスクロールアニメーション

## コンテンツ

1. **パワーストーンとは** - 石の意味と価値
2. **浄化方法** - 月光浴、水晶クラスター、流水、セージ、音叉
3. **お手入れ方法** - 日常のケアと保管方法
4. **効果を高める使い方** - 身につける、瞑想、空間に置く、アファメーション

## 技術スタック

- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Vite

## セットアップ

```bash
pnpm install
pnpm dev
```
